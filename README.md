## Περιγραφή

Η εργασία αφορά την ανάπτυξη και προσομοίωση ενός βιομηχανικού δικτύου παρακολούθησης αισθητήρων με χρήση Cisco Packet Tracer.

Το δίκτυο περιλαμβάνει:

- routers
- switches
- υπολογιστές
- web server

Η δρομολόγηση πραγματοποιείται με χρήση OSPF.

---

## Λειτουργίες

Το δίκτυο υποστηρίζει:

- επικοινωνία μεταξύ όλων των συσκευών
- δυναμική δρομολόγηση OSPF
- ping tests
- HTTP service
- web page hosting
- HTTP request από client προς server

## Σημείωση

Η πλήρης web εφαρμογή PHP/MySQL αναπτύχθηκε ξεχωριστά σε περιβάλλον XAMPP.

Σε ορισμένα βοηθητικά σημεία έγινε περιορισμένη χρήση εργαλείων Τεχνητής Νοημοσύνης για υποστήριξη στην οργάνωση και τεκμηρίωση της εργασίας.
